##  A sources codes React Native Tutorial #1

This sources codes aims to get you up to speed with writing iOS apps using React Native. If you're wondering what React Native is and why Facebook built it, this [blog post](https://code.facebook.com/posts/1014532261909640/react-native-bringing-modern-web-techniques-to-mobile/) explains that.

### Setup

React Native requires OSX, Xcode, Homebrew, io.js , npm (installed automatically with io.js), and watchman.

```bash
$ npm install -g react-native-cli
$ git clone <this project>
$ cd <this project>
$ open AwesomeProject.xcodeproj # (on osx)
$ vim .
```
